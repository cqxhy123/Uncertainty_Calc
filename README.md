# 合成不确定度计算器

一个用于大学物理实验数据处理的工具，支持 **A 类不确定度、B 类不确定度计算** 及 **多分量合成不确定度**，提供两种使用方式：
- **命令行版（C++）**：适合批量计算或集成脚本，交互式输入。
- **网页版（HTML）**：双击在浏览器里打开，直观方便，无需安装任何环境。

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `main.cpp` | 命令行版不确定度计算程序（C++） |
| `index.html` | 网页版计算器，纯前端实现，双击即用 |

## 🚀 快速开始

### 方式一：使用网页版（推荐，零门槛）

1. 下载 `index.html` 文件（或克隆整个仓库）。
2. 直接**双击** `index.html`，用默认浏览器打开。
3. 按照界面提示输入数据，点击相应按钮即可计算结果。
   - A 类不确定度：输入多次测量值，自动计算平均值和 A 类不确定度。
   - B 类不确定度：输入仪器误差 delta，按矩形分布计算 uB = delta/√3。
   - 手动合成：手动输入 uA 和 uB，计算合成不确定度。
   - 表格合成：多分量不确定度合成，支持自定义系数。

> 网页版所有计算均在你的浏览器中完成，**无需网络、无需安装**。

### 方式二：使用命令行版（需要 C++ 编译环境）

如果你需要命令行自动化、批量处理，或者想查看源代码学习，可以使用此版本。

#### 编译
确保已安装 g++ 编译器（MinGW/Linux/macOS 均可）。
